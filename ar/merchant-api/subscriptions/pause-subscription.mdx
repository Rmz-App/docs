# إيقاف الاشتراك مؤقتًا

<div className="flex items-center gap-2 mb-4">
  <span className="px-2 py-1 bg-blue-100 text-blue-800 rounded text-sm font-mono font-bold">POST</span>
  <code>/subscriptions/{"{id}"}/pause</code>
</div>

يوقف اشتراكًا نشطًا بصورة مؤقتة. يُعلَّق التجديد التلقائي وتصبح قيمة `is_active` مساوية لـ `false`. الوصول معلّق أثناء الإيقاف المؤقت، والأيام المتبقية من الفترة تُحفظ تلقائيًا داخليًا وتُعاد عند استئناف الاشتراك.

## الترويسات (Headers)

| الترويسة | القيمة | مطلوبة |
|--------|-------|----------|
| `Authorization` | `Bearer YOUR_API_TOKEN` | نعم |
| `Accept` | `application/json` | مُستحسنة |

## معاملات المسار (Path Parameters)

| المعامل | النوع | مطلوب | الوصف |
|-----------|------|----------|-------------|
| `id` | integer | نعم | معرّف الاشتراك |

## استجابة النجاح

يُرجع كائن الاشتراك الكامل بالشكل القياسي `SubscriptionResource` — مطابق لكلٍّ من [عرض اشتراك](./get-subscription) و[قائمة الاشتراكات](./list-subscriptions) و[البحث عن الاشتراكات](./lookup-subscriptions). فيما يلي أكثر الحقول صلة بالإيقاف المؤقت — راجع [عرض اشتراك](./get-subscription) للاطلاع على قائمة الحقول الكاملة.

```json
{
  "message": "Subscription paused successfully",
  "data": {
    "id": 501,
    "status": "paused",
    "external_customer_id": "usr_abc123",
    "is_active": false,
    "current_period_end": "2025-07-01T00:00:00.000000Z"
  },
  "api": "rmz.shawarma",
  "timestamp": 1699999999
}
```

## السلوك

- لا يمكن إيقاف إلا الاشتراكات التي حالتها `active`
- يُعلَّق التجديد التلقائي — ولن تُجرى أي محاولات دفع
- تُضبط `is_active` على `false`
- الوصول معلّق أثناء الإيقاف المؤقت، والأيام المتبقية من الفترة تُحفظ تلقائيًا في الـ metadata
- عند الاستئناف، تُمدَّد الفترة بعدد الأيام المتبقية المحفوظة
- يُرسَل حدث Webhook باسم `subscription.paused`

<Note>
  إيقاف الاشتراك مؤقتًا لا يسترد أي مبالغ مدفوعة. فهو يوقف التجديد التلقائي فقط ويضع الاشتراك في حالة غير نشطة. استخدم هذه الميزة للإيقافات المؤقتة لا للإلغاءات.
</Note>

## استجابات الأخطاء

| الرمز | الوصف |
|------|-------------|
| `401` | غير مُصرّح — رمز الوصول غير صالح أو مفقود |
| `404` | الاشتراك غير موجود |
| `422` | لا يمكن إيقاف إلا الاشتراكات النشطة |
