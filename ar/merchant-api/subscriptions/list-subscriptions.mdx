# عرض الاشتراكات

<div className="flex items-center gap-2 mb-4">
  <span className="px-2 py-1 bg-green-100 text-green-800 rounded text-sm font-mono font-bold">GET</span>
  <code>/subscriptions</code>
</div>

تُعيد هذه النقطة قائمةً مُرقَّمة باشتراكات العملاء الخاصة بالمتجر المُصادَق عليه. يمكنك التصفية حسب البريد الإلكتروني للعميل أو رقم هاتفه أو رمز دولته.

## الترويسات (Headers)

| الترويسة | القيمة | مطلوبة |
|--------|-------|----------|
| `Authorization` | `Bearer YOUR_API_TOKEN` | نعم |
| `Accept` | `application/json` | مُستحسن |

## معاملات الاستعلام (Query Parameters)

| المعامل | النوع | مطلوب | الوصف |
|-----------|------|----------|-------------|
| `page` | integer | لا | رقم الصفحة (الافتراضي: 1) |
| `customerEmail` | string | لا | التصفية حسب البريد الإلكتروني للعميل |
| `customerPhone` | string | لا | التصفية حسب رقم هاتف العميل |
| `customerCountryCode` | string | لا | التصفية حسب رمز دولة العميل (مثل `966`) |

## مثال على الطلب

<CodeGroup>
```bash cURL
curl -X GET "https://merchant-api.rmz.gg/shawarma/subscriptions?customerEmail=ahmed@example.com" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```javascript JavaScript
const params = new URLSearchParams({
  customerEmail: "ahmed@example.com"
});

const response = await fetch(
  `https://merchant-api.rmz.gg/shawarma/subscriptions?${params}`,
  { headers: { "Authorization": "Bearer YOUR_API_TOKEN" } }
);
const subscriptions = await response.json();
```

```python Python
import requests

response = requests.get(
    "https://merchant-api.rmz.gg/shawarma/subscriptions",
    headers={"Authorization": "Bearer YOUR_API_TOKEN"},
    params={"customerEmail": "ahmed@example.com"}
)
subscriptions = response.json()
```

```php PHP
$params = http_build_query(["customerEmail" => "ahmed@example.com"]);
$ch = curl_init("https://merchant-api.rmz.gg/shawarma/subscriptions?{$params}");
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "Authorization: Bearer YOUR_API_TOKEN"
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = json_decode(curl_exec($ch), true);
```
</CodeGroup>

## قيم المدة (Duration)

يمثّل الحقل `duration` في كل اشتراك قيمةً مُعدَّدة (enum) تُعبّر عن فترة الفوترة لمتغيّر الاشتراك. القيم المُتاحة:

| القيمة | المعنى |
|-------|---------|
| `monthly` | شهر واحد |
| `quarterly` | 3 أشهر |
| `semiAnnual` | 6 أشهر |
| `annually` | سنة واحدة |
| `biennial` | سنتان |
| `quinquennial` | 5 سنوات |
| `decennial` | 10 سنوات |

## قيم الحالة (Status)

يعكس الحقل `status` الحالة الحالية لدورة حياة الاشتراك. القيم المُحتملة:

| القيمة | المعنى |
|-------|---------|
| `trialing` | خلال فترة تجريبية؛ الوصول مُتاح |
| `active` | مدفوع بالكامل ونشِط؛ الوصول مُتاح |
| `past_due` | فشل دفع التجديد؛ قد يكون ضمن فترة السماح |
| `paused` | مُوقَف مؤقتًا من قِبَل التاجر أو العميل |
| `canceled` | مُلغى؛ قد يبقى الوصول حتى نهاية الفترة الحالية |
| `expired` | حالة نهائية؛ لا يُتاح الوصول |

## أمثلة على التصفية

### حسب هاتف العميل

```bash
curl -X GET "https://merchant-api.rmz.gg/shawarma/subscriptions?customerCountryCode=966&customerPhone=501234567" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

### جميع الاشتراكات (بدون تصفية)

```bash
curl -X GET "https://merchant-api.rmz.gg/shawarma/subscriptions?page=1" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```
